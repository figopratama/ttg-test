## How to Run Each Task

### Task 1 — Frontend
Open `1-frontend/index.html` in any browser. No installation needed.

### Task 2 — Backend API
```bash
cd 2-backend
npm install
# Create .env file with your MongoDB URI
node server.js
```

### Task 3 — Problem Solving
```bash
cd 3-number-finding
python3 find_missing.py
```

### Task 4 — Calculation Formula
```bash
cd 4-calculation-formula
python3 calculator.py
```
