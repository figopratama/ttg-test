# Task 1 — Front-End: User Registration Form

A simple, mobile-friendly user registration form built with plain HTML, CSS, and JavaScript. No frameworks, no build tools, no dependencies.

## Features

- Fields: Full Name, Email, Password, Password Confirmation
- Email domain validation (gmail.com, yahoo.com, outlook.com, etc.)
- Password minimum 8 characters with live strength indicator
- Password confirmation match check
- Live inline validation with visual feedback
- "Pendaftaran berhasil" success message on valid submission

## How to Run

1. Clone the repository:
   ```bash
   git clone <repo-url>
   ```

2. Navigate to this folder:
   ```bash
   cd 1-frontend
   ```

3. Open `index.html` in any modern browser:
   - **Windows:** double-click `index.html`, or run:
     ```bash
     start index.html
     ```
   - **Mac:**
     ```bash
     open index.html
     ```
   - **Linux:**
     ```bash
     xdg-open index.html
     ```

No installation, server, or build step required.
